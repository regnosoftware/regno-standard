Support - GitHub

The latest version of the official documentation can be found on this site, with full version control and a historical change log available on our public GitHub site.
GitHub Issues are preferred for discussion of the Regno Standard. When filing an issue, please put the text “Regno Standard Specification” in the title, preferably like this: “[Regno Standard Specification] …summary of comment…”.
All issues and comments are archived, and there is also a historical archive.

Support - Governance

The Regno Standard is governed by the Regno Standard Working Group. If you would like more details about the Regno Standard Working Group or would like to become involved, please contact the Working Group here.
For initial technical support the group may be contacted directly here.

Support - Commercial

Commercial Support is provided by Regno Software, home of the official API implementation of the Regno Standard.
Regno Software also provides tools and a platform that support the Regno Standard, as well as product support options. Please contact Regno Software directly for assistance in implementing the Regno API.
