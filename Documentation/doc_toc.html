  5. Configuration Documents
    1. ConfigDoc

        The ConfigDoc document provides an overview of the type and structure of data contained in the child documents, it also acts as a basic meta data container to describe the data stored by this configuration.

    2. TimeSpanDoc

        The TimeSpanDoc document is a data document that contains the properties and a data values for an individual time spand instance.

  6. Identity Documents
    1. IdentityDoc

        The IdentityDoc document is an optional identity document that allows users to define ownership and include identifying markers for a collection of documents. If used, the IdentityDoc document can be referenced by all other documents.

  7. Informational Documents
    1. ConfigDefinitionsDoc

        The ParamFrequencies document is an informational document that defines the collection frequencies that are contained for a given Parameter Definition Document Collection.

    2. ConfigParamFrequnciesDoc

        The ConfigParamFreqDoc document is an informational document that defines the parameters logged within the context of a config doc and their related logged frequencies.

  8. Definition Documents
    1. ParamDefinitionDoc

        The CANParamDefinitionDoc document is a definition document that contains the properties for a CAN parameter instance.

    2. EventDefinitionDoc

        The EventDefinitionDoc document is a definition document that contains the properties for a unique event definition instance.

    3. CANBusDefinitionDoc

        The CANBusDefinitionDoc document is a definition document that contains the properties for a CAN bus instance.

    4. CANBufferDefinitionDoc

        The CANBufferDefinitionDoc document is a definition document that contains the properties for a CAN buffer instance.

    5. CANMessageDefinitionDoc

        The CANMessageDefinitionDoc document is a definition document that contains the properties for a CAN message instance.

    6. CANParamDefinitionDoc

        The CANParamDefinitionDoc document is a definition document that contains the properties for a CAN parameter instance.

    7. TriggerDefinitionDoc

        The TriggerDefinitionDoc document is a definition document that contains the properties for a trigger instance.

    8. TriggerConditionDefinitionDoc

        The TriggerConditionDefinitionDoc document is a definition document that contains the properties for a trigger condition instance.

  9. Data Documents
    1. ChannelSamplesDoc

        The ChannelSamplesDoc document is a data document that contains the properties and values for a collection of sub or super sampled data samples.

    2. ParamSamplesDoc

        The ParamSamplesDoc document is a data document that contains the properties and values for a collection of data samples for a single data channel.

    3. ParamStatDoc

        The ParamStatDoc document is a data document that contains the properties and statistics values for a parameter instance over a specified time range.

    4. ParamScalarValueDoc

        The ParamScalarValueDoc document is a data document that contains the properties and a set of values for a single configuration parameter.

    5. ParamStringValueDoc

        The ParamStringValueDoc document is a data document that contains the properties and a set of values for a single configuration parameter.

    6. ParamArrayValueDoc

        The ParamArrayValueDoc document is a data document that contains the properties and a set of values for a single configuration parameter.

    7. Param2DMapValueDoc

        The Param2DMapValueDoc document is a data document that contains the properties and a set of values for a single two dimensional map parameter.

    8. Param3DMapValueDoc

        The Param3DMapValueDoc document is a data document that contains the properties and a set of values for a single three dimensional map parameter.

    9. EventDataDoc

        The EventDataDoc document is a data document that contains the properties and a data values for an individual event instance.

    10. MediaDataDoc

        The MediaDataDoc document is a data document that contains the properties and a source information for a media data file.

    11. StatDataDoc

        The StatDataDoc document is a data document that contains the properties and a source information for a statistic data file.

    12. doc_data_tagDataDoc.html

        The doc_data_tagDataDoc.html document is a data document that contains the key value pair of a tag.
