Documentation - Overview

Regno Standard documents are a collection of plain text documents using the industry standard JSON format. Each of the REGNO STANDARD documents provides a subset of functionality that when combined provide a rich specification to handle all data types irrespective of format or source.


Documentation - Status

Regno Software has now officially release version 1.0 of the Regno Standard as the official reference documentation. The ongoing maintenance and development of the Regno Standard will be carried out by
Regno Software under the oversight of the Regno Open Data Standard Working Group (hereon known as the Working Group).

GitHub Issues are preferred for discussion of this specification. Please see the Support section for more details.
